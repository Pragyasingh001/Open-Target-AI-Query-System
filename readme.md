# Biomedical AI Query System

An AI-powered system that converts natural language queries into structured GraphQL queries to retrieve disease–target associations from biomedical datasets using the Open Targets platform.

Overview
This project enables users to ask biomedical questions in natural language and automatically translates them into valid GraphQL queries to fetch accurate results from APIs.

Key Features
- Natural Language → GraphQL query conversion  
- Tool-driven query generation and execution  
- Structured prompt engineering for consistent outputs  
- Automated response parsing for clean and readable results  
- Reliable and non-hallucinated data retrieval from APIs

Tech Stack
- Python  
- Open Targets GraphQL API  
- LLM (for query generation)
